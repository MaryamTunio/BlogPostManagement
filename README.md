# Blog Post Management Interface

A responsive Blog Post Management Interface built using HTML, CSS, and JavaScript. The application allows users to create, edit, preview, and delete blog posts while storing data in the browser's localStorage for persistence.

## Features

* Create new blog posts
* Edit existing posts
* Delete blog posts
* Real-time live preview while typing
* Form validation for required fields
* Responsive design for desktop and mobile devices
* LocalStorage integration for data persistence
* Clean card-based post display

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Browser LocalStorage

## How It Works

1. Enter a title, author name, publication date, and content.
2. View changes instantly in the Live Preview section.
3. Click "Publish Post" to save the post.
4. Posts are displayed below in a card layout.
5. Edit or delete posts anytime.
6. Data remains available after page refresh using localStorage.
